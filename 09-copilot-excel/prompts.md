# 09 — Copilot in Excel: Prompts

Open `AI_Guideline_Survey_Responses.xlsx` from this folder. Format the data as an Excel Table (Ctrl+T), then open **Copilot** from the Home ribbon.

---

## Getting a quick overview

```
Give me a summary of this survey data. What are the key findings about how employees are using AI tools and how they feel about the AI Usage Guideline?
```

```
What are the top 3 insights from this dataset I should highlight to management?
```

---

## Department analysis

```
Which departments gave the highest average productivity ratings? Rank them from highest to lowest.
```

```
Are there any departments that seem significantly less confident about using AI compared to others? Show me the difference.
```

```
Compare the average productivity rating against the average confidence rating for each department. Which department has the biggest gap between the two?
```

---

## Creating charts

```
Create a bar chart showing how often employees refer to the AI Usage Guideline, broken down by frequency: Never, Rarely, Sometimes, Often, Always.
```

```
Create a chart comparing average productivity rating by department.
```

```
Create a pie chart showing which Copilot app is used most across all respondents.
```

---

## Finding patterns

```
Is there a pattern between how often employees reference the guideline and their confidence rating? Do employees who refer to it more often feel more confident?
```

```
Which respondents gave the lowest NPS scores (0-6)? What department are they from and what did they say in their open feedback?
```

```
How many respondents said the guideline has NOT changed their approach to using AI? What do they have in common?
```

---

## Formula help

```
Write a formula to calculate the percentage of respondents who gave a productivity rating of 4 or 5.
```

```
Write a COUNTIF formula to count how many people use Copilot Chat as their most-used app.
```

```
Explain what this formula does in plain language: =AVERAGEIF(C2:C31,"Finance",G2:G31)
```

---

## Writing the management summary

```
Based on this data, write a 120-word management summary of the survey results. Include the overall NPS, average productivity rating, top positive finding, and one recommended action. Write it in a tone suitable for a department head report.
```

```
Based on the open feedback in columns M and N, what are the 3 most common themes in what employees say is working well? And the 3 most common improvement suggestions?
```
