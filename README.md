## How to Use This Site

Each topic has a **Notes** page covering the concepts and a **Prompts** page with copy-paste prompts you can use during and after the session.

To copy a prompt: open the prompts file, find the prompt you want, and click the copy icon on the top right of the code block.

To download everything: click the green **Code** button on the main repo page, then **Download ZIP**.

## Workshop Scenario

Throughout this workshop you work on one connected project from start to finish.

> **You are creating an AI Usage Guide for your department.**

This guide will explain how your team should use AI tools responsibly and effectively. Each topic moves the project one step forward:

| Stage | What happens |
|-------|-------------|
| Research | Use Copilot Chat to research AI best practices, regulations, and department-specific use cases |
| Draft | Move your research into Copilot Pages and build the first draft of your guide |
| Refine | Open the draft in Word and reformat it as a formal management proposal |
| Submit | Email the proposal to your supervisor for approval using Copilot in Outlook |
| Present | Turn the approved document into a team presentation using Copilot in PowerPoint |
| Survey | Build a knowledge quiz and feedback survey using Copilot in Forms |
| Analyse | Use Copilot in Excel to analyse the quiz results and surface insights |
| Automate | Build a simple weekly check-in agent in Copilot Studio to keep the feedback loop running |

| 09 | [Copilot in Excel](./09-copilot-excel/) | [Prompts](./09-copilot-excel/prompts) | Analyse quiz results, add calculated columns, identify weak areas by department, and use Python in Excel to generate charts |
| 10 | [Copilot Studio Intro](./10-copilot-studio-intro/) | [Prompts](./10-copilot-studio-intro/prompts) | Build a simple weekly check-in agent that asks your team one question about the AI guideline and collects their response |
