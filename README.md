# Microsoft 365 Copilot Workshop

Prompts, reference notes, and sample datasets for the Microsoft 365 Copilot workshop.

---

## How to Use This Repo

Each folder covers one topic. You can work through them in order or jump to whichever section matches where we are in the session. Everything here is yours to keep, copy, and adapt after the workshop.

To copy a prompt: open the file, find the prompt you want, and click the copy icon on the top right of the code block.

To download everything: click the green **Code** button on the main repo page, then **Download ZIP**.
