# 05 — Copilot in Word: Prompts

Use these in **Copilot in Word** to turn your draft into a formal management proposal.

---

## Reformatting as a proposal

```
Reformat this document as a formal proposal to submit to management. Add these sections if missing:
- Executive Summary (max 150 words)
- Business Justification (why this guideline is needed now)
- Implementation Steps (what needs to happen to roll this out)
- Expected Outcomes (what the organisation will gain)
- Approval and Sign-off section

Keep all existing content but adjust the tone to be formal and suitable for senior management.
```

---

## Strengthening specific sections

```
Rewrite the Executive Summary in 3 sentences only: one sentence stating the problem, one stating the proposed solution, one stating the expected benefit.
```

```
Add 2 specific, realistic examples of AI-related risks that Malaysian companies have faced or could face if employees use AI without guidelines. Keep it factual and brief.
```

```
Add a new section called "Risk of Not Having This Guideline" with 3 consequences written in professional business language. Not alarmist — factual.
```

---

## Tone and style adjustment

```
This document reads as too informal for a management submission. Rewrite it using formal business English throughout while keeping it easy to scan.
```

```
The Dos and Don'ts section currently reads as a rules list. Rewrite it as a set of guided principles — each point should include a one-sentence rationale so employees understand why the rule exists.
```

---

## Cross-referencing other documents

If you have your company's IT or HR policy on OneDrive, use the `/` key to reference it:

```
Using the [company IT policy document], check whether any recommendations in this AI Usage Guide conflict with existing IT policy. List any inconsistencies.
```

---

## Final polish

```
Review the entire document for formatting consistency. Make sure all headings use the same style, all bullet points are parallel in structure, and the tone is consistent throughout.
```

```
Create a one-page cover summary for this proposal. Include: document title, purpose in 2 sentences, 3 key recommendations as bullet points, date, and a sign-off block with spaces for name, designation, and signature.
```

---

## Using Track Changes

Before sending: turn on **Track Changes** (Review tab), then ask Copilot to make one more revision so you can compare the before and after.
