# 02 — Prompt Engineering: Prompts

Practice writing better prompts using the **GCSE framework** (Goal, Context, Source, Expectations).

---

## Rewriting weak prompts

Take each weak prompt below and rewrite it using GCSE before submitting:

**Weak:**
```
Write me something about AI.
```

**Weak:**
```
Summarise this.
```

**Weak:**
```
Help me with my email.
```

---

## GCSE framework practice

```
I am an HR manager at a 200-person Malaysian manufacturing company. Write a short internal announcement (under 120 words) informing staff that the company has adopted Microsoft 365 Copilot and will be rolling out training next month. Tone: reassuring and positive.
```

```
Act as a sceptical finance director reviewing a new AI Usage Guideline. List 5 questions or concerns you would raise before approving it. Be specific and practical.
```

```
I need to explain to a non-technical colleague what "prompt engineering" means. Write a 3-sentence explanation using an analogy from everyday life in Malaysia.
```

---

## Iteration practice

Submit this first prompt, then use the follow-up prompts to refine the response:

**Round 1:**
```
Write a short policy statement on the responsible use of AI at work.
```

**Round 2 (follow-up):**
```
Make it more specific to Malaysian workplace culture. Add a reference to PDPA (Personal Data Protection Act).
```

**Round 3 (follow-up):**
```
Shorten it to under 80 words and make it suitable for a company intranet homepage.
```

---

## Format control

```
List 5 benefits of using Copilot in Outlook. Give the output in 3 different formats: (1) bullet points, (2) a numbered list with one-line explanations, (3) a table with "Benefit" and "Time Saved" columns.
```

---

## Persona and perspective

```
Act as a cautious IT security officer. Review this sentence from an AI Usage Guideline and flag any security risks: "Staff may use any AI tool of their choice for work tasks as long as it improves productivity."
```
